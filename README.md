# signal-desk-web

`SignalDesk` 프론트엔드(React + Vite).

## 1차 릴리즈 범위
- `시장` 탭
- 한국/미국 분리
- 한국/미국 장 상태
- 정규장/장전/장후/휴장
- 지수 차트(일별/월별/연별)
- 캔들 + 거래량 + 이동평균선(5/20/60)
- 수급(개인·외국인·기관)
- 공포/열기 지표
- 뉴스 군집화 + 시장 요약
- `종목` 탭
- 대표 종목 + 검색
- 종목 상세 패널(연관 뉴스/관심/보유/AI 추천 상태)
- 관심종목 직접 추가/삭제
- `포트폴리오` 탭
- 보유 종목 손익/평가
- 포트폴리오 직접 추가/삭제
- `AI 추천` 탭
- 추천 종목 + 추천 성과
- 추천 종목과 연관 뉴스 연결
- 추천 근거/성과 로그 타임라인
- `모의투자` 탭
- 모의 보유종목/거래 로그 직접 관리

## 성능 전략
- 초기 진입은 `summary`, `sections`, `news`만 호출
- 나머지 탭은 전환 시 지연 로딩
- 저장/삭제 후 해당 탭 데이터만 재호출
- 차트 모듈은 시장 탭 `지수와 차트` 진입 시점에 lazy import
- 차트는 `MarketIndexCharts` 청크로 분리되어 초기 로딩과 분리

## 실행
```bash
npm install
npm run dev
```

## 환경변수
- 로컬 개발 서버에서 API 직결 시
```bash
VITE_API_BASE_URL=http://localhost:8091
```
- Docker(`signal-desk-web` nginx 프록시) 사용 시 기본값(`/api`)으로 동작
